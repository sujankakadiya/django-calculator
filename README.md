# 🧮 Django Calculator

A simple calculator web application built using Django.
It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🎨 Clean and simple UI
* ⚡ Fast calculation

---

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/sujankakadiya/django-calculator.git
```

2. Navigate to project folder:

```
cd django-calculator
```

3. Run the server:

```
python manage.py runserver
```

4. Open in browser:

```
http://127.0.0.1:8000/
```

## 💡 How It Works

* User enters two numbers
* Selects an operator (+, -, *, /)
* Clicks calculate
* Result is displayed instantly
